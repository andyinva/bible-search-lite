──────────────────────────────────────

  Then: Copy win_install folder to USB

───────────────────────────────
  Run: INSTALL.bat (as Administrator)
  OR
  Run: 02_install_offline.ps1 (as Administrator)


REQUIREMENTS
════════════

Download Phase:
  ✓ Windows 10/11
  ✓ Internet connection
  ✓ PowerShell 5.1+

Install Phase:
  ✓ Windows 10/11 (offline OK!)
  ✓ Administrator privileges
  ✓ 500 MB free disk space
  ✓ USB drive with downloaded files


WHAT GETS INSTALLED
═══════════════════

✓ Python 3.12.1 (~200 MB)
✓ PyQt6 GUI framework (~50 MB)
✓ Bible Search Lite application (~50 MB)
✓ Desktop shortcut

Total: ~300 MB

COMMON QUESTIONS
════════════════

Q: Do I need internet on the target PC?
A: No! That's the whole point of this offline installer.

Q: Will this affect my existing Python?
A: No! This is a completely isolated installation.

Q: How long does it take?
A: Download: 5-10 min | Install: 5 min | Total: ~15 min

Q: Can I delete the USB drive after installing?
A: Yes! Keep it as a backup though.
