## Troubleshooting

### HTML files are outdated
**Solution:** Run `python3 convert_to_html.py` again

### index.html won't open
**Solution:** Check file permissions, try opening from browser's File menu

### Styling looks broken
**Solution:** Make sure all files are in the same folder, refresh browser

### Can't find a document
**Solution:** Check if the `.md` file exists, re-run converter
