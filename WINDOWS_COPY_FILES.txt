# Files to Copy to Windows 11 for Testing
# =========================================

## Method 1: Manual Copy (Easiest)
Just copy these files/folders from your WSL to Windows:

### Main Directory Files:
bible_search_lite.py
bible_search.py
bible_search_service.py
subject_manager.py
subject_verse_manager.py  
subject_comment_manager.py
VERSION.txt
README.md
SEARCH_OPERATORS.md
WINDOWS_INSTALL.md
LICENSE

### Folders (copy entire folders):
bible_search_ui/
database/

### Create Launcher:
Create a file called "run_bible_search.bat" with this content:
@echo off
echo Starting Bible Search Lite...
python bible_search_lite.py
pause
